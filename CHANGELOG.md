# Changelog

## [1.0.0] - 2026-03-10

### Added
- Next.js App Router architecture for Vercel deployment.
- Story library with search/sort and reader-first detail page.
- Persistent reviews API (`GET`/`POST`/`DELETE`) with Upstash Redis adapter.
- Local file-based persistence fallback for development.
- Reader tools: font size, line length, saved reading position, and progress display.
- Security basics: payload validation, sanitization, rate limiting, moderation flagging.
- Repository standards: header validator script and CI workflow.

### Compliance
- ZPL v2.0 license file retained at repository root.
